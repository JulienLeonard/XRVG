Attributable module

Allows class attribute definition in a concise and effective way. Can be viewed as a :attr_accessor extension. See also Attribute

Example : 

  class A
     attribute :a 1.0; # type is then Float
     attribute :b      # attribute is required
  end

  a = A[ :a, 10.0, :b, 2.0 ]
  a.a => 10.0

[Validate]
