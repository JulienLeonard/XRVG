Class Line
In: lib/shape.rb
Parent: Curve

Line class

Intro

Used to draw polylines and polygons

Attributes

 attribute :exts, [Point[0.0, 0.0], Point[1.0, 1.0]]

Example

 line = Line[ :exts, [Point::O, Point::X] ]
