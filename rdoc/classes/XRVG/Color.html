Class XRVG::Color
In: lib/color.rb
Parent: Object

Color class

Basics

Class Color consists in a 4D vector of (0.0..1.0) values, for red, blue, green, and opacity

Utilities

Conversion from hsv and hsl color spaces available (see this link)

Future

  • Must use this library, to avoid effort duplication
  • Must add relative color operations as Nodebox wants to
  • Must optimize 4D vector operations (as C extension ?)

hsl( h, s, l)
hsla( h, s, l, a)

build a color vector from hsl parametrization (convert from hsl to rgb) h, s, l being between 0.0 and 1.0 taken from [[en.wikipedia.org/wiki/HSV_color_space]] h, s, l must be between 0.0 and 1.0

hsv( h, s, v)
hsva( h, s, v, a)

build a color vector from hsv parametrization (convert from hsv to rgb) h, s, v being between 0.0 and 1.0
